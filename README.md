# Echo Chatbot

Echo is a Streamlit-based mental health support chatbot with a Zapier Interfaces chatbot embedded in the UI.

## Run locally

```powershell
pip install -r requirements.txt
streamlit run Echo/streamlit_app.py
```

## Configure Zapier Interfaces

The Chatbot tab embeds a Zapier Interfaces chatbot using the `zapier-interfaces-chatbot-embed` web component.
